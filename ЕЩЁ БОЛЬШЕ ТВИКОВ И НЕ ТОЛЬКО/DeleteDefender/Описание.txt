DefenderKiller - утилита, которая позволяет узнать текущее состояние компонентов Защитника Windows и безвозвратно удалить его.

Для запуска утилиты просто дважды нажмите на Delete Windows Defender.bat.

ВАЖНО. Для более полного удаления Защитника, желательно выполнить проходку в несколько раз, то есть пройти весь процесс удаления 2 раза.
