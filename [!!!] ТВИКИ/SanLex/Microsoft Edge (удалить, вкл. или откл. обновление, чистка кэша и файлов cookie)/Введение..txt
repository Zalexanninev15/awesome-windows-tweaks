Средство удаления Microsoft Edge — это инструмент, призванный помочь пользователям Windows удалить Microsoft Edge Chromium и Edge UWP со своих устройств.
Дополнительные скрипты для отключения или включения обновления браузера Edge

Руководство:
Выбирать предпочтительный скрипт > запустить от имени Администратора > после завершения процесса закрыть окно > перезагрузить ПК, чтобы изменение вступили в силу.

Удалить по пути: C:\Users\%username%\AppData\Local\Microsoft\Edge

Удалить по пути: C:\ProgramData\Microsoft\EdgeUpdate
